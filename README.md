Stripe API Trailpack for Trails

## Install

```sh
$ npm install --save trailpack-stripe
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-stripe')
  ]
}
```
